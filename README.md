# Ruta Digital para tu MYPE

Sitio estático listo para GitHub Pages. No requiere backend, base de datos ni claves API.

## Publicación rápida en GitHub Pages

1. Sube estos archivos a un repositorio:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `assets/hero.png`
2. En GitHub, entra a `Settings > Pages`.
3. En `Build and deployment`, elige `Deploy from a branch`.
4. Selecciona la rama principal y la carpeta donde estén estos archivos.
5. Guarda los cambios y espera a que GitHub publique la URL.

## Personalización

Puedes cambiar el texto visible en `index.html`, los colores en `styles.css` y las preguntas o reglas del diagnóstico en `app.js`.
