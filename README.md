# Ruta Digital para tu MYPE

Aplicación web interactiva para ayudar a emprendedores, administradores y dueños de pequeñas empresas a construir un plan práctico de transformación digital.

Creado por **Ing. Segundo Morales**.

## ¿Qué hace esta herramienta?

La herramienta guía al usuario mediante un diagnóstico simple sobre su negocio, su nivel de digitalización, sus principales problemas y sus objetivos.
Al finalizar, genera una ruta de transformación digital con recomendaciones prácticas para mejorar ventas, atención al cliente, finanzas, inventario, operaciones, administración y toma de decisiones con datos.

## ¿Para quién está pensada?

Está pensada para MYPES, pequeños negocios, emprendedores y responsables de negocios que desean empezar a ordenar sus procesos usando herramientas accesibles y de bajo costo.
No se requiere conocimiento técnico avanzado.

## Funcionalidades principales

- Diagnóstico guiado por bloques de preguntas.
- Evaluación del nivel de madurez digital.
- Identificación de principales brechas y oportunidades.
- Roadmap de transformación digital de 6 o 12 meses.
- Recomendaciones de herramientas gratuitas, intermedias y avanzadas.
- Indicadores mínimos para controlar el negocio.
- Ideas de automatización simple.
- Prompts prácticos para usar inteligencia artificial.
- Opción para copiar o descargar el plan final.

## Tecnologías utilizadas

Este proyecto está construido con tecnologías web simples:

- HTML
- CSS
- JavaScript

No requiere backend, base de datos ni instalación de dependencias.

## Cómo usarlo

1. Abre la página web.
2. Presiona **Iniciar diagnóstico**.
3. Responde las preguntas sobre tu negocio.
4. Al finalizar, revisa tu plan personalizado.
5. Copia o descarga el resultado.
